The Code

Form inputs should be enclosed inside an element with class "form-field" for styled inputs. Omit the "form-field" containing element for a default input with no styling.

Inline error styling is achieved with the addition of the class "form-field--error" to the element with class "form-field" and by adding the error message content to the form label.

// Disabled
<div class="form-field form-field--disabled">
<input type="text" name="text" id="text" autocomplete="off" disabled/>
<label for="email">Hint Label Text</label>
</div>

// Active
<div class="form-field">
<input type="email" name="email" id="email" autocomplete="off" />
<label for="email">Hint Label Text</label>
</div>

// Inline Error
<div class="form-field form-field--error"> <input type="email" name="email2" id="email2" autocomplete="off" /> <label for="email2">Hint Label Text:</label> </div>
